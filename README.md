# 🌟 HabitSpark — 学习习惯成长记

> 专为小学生打造的积分制学习习惯养成系统

## ✨ 项目简介

HabitSpark（学习习惯成长记）是一个面向学龄儿童和家长的家庭教育辅助工具。系统围绕**课前预习、课后复习、作业规范、错题整理、自主阅读、时间管理**六大核心学习习惯，结合孩子的兴趣爱好（唱歌、手工、漫画、游乐场），通过**正向积分激励机制**促进习惯养成。

## 👥 用户角色

| 角色 | 说明 |
|------|------|
| 👧 学生 | 上海四年级下学期 11 岁女生，完成任务打卡、查看积分、兑换礼品 |
| 👨‍👩‍👧 家长 | 负责任务审核、积分管理、商品管理、数据分析 |

## 🏗️ 技术栈

### 后端
- **框架**: Spring Boot 3.x (Java 17)
- **数据库**: SQLite 3.45+
- **ORM**: MyBatis-Plus
- **认证**: JWT

### 前端
- **框架**: Vue 3 + Vite + Pinia
- **学生端**: 自定义卡通 UI 组件
- **家长端**: Element Plus 管理后台

## 🚀 快速开始

### 环境要求
- Java 17+
- Node.js 18+
- npm / pnpm

### 后端启动
```bash
cd backend
./mvnw spring-boot:run
```

### 前端启动
```bash
cd frontend
npm install
npm run dev
```
